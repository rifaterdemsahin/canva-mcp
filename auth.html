How to authenticate

  1. Connect — Click "Connect to Canva" below. You'll be redirected to Canva's authorization page as info@pexabo.com.
  2. Allow — Review the permissions (designs, assets, folders) and click the purple Allow button.
  3. Callback — Canva redirects back here with an authorization code (green box). The page tries to auto-exchange it, but Canva's API requires the client secret.
  4. Get secret — Open terminal and run: 5_Symbols/toolbox/secrets.sh get canva-mcp-CANVA-CLIENT-SECRET
  5. Exchange — Paste the secret into the CLIENT SECRET field below and click Exchange Tokens.
  6. Save token — Copy the access token (automatic). The MCP server reads it from .env at CANVA_ACCESS_TOKEN.
